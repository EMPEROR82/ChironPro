# Chiron Extension

VS Code extension for running Chiron (.tl) programs.

## Features
- Run `.tl` files directly from editor
- Webview-based turtle rendering
- Language support (syntax + snippets)

## Requirements
- Python
- uv (https://github.com/astral-sh/uv)

## Usage
Open a `.tl` file and press **F5** or click **Run Chiron File**.
